# Rules Information Layout

Store pulled Wizards Comprehensive Rules text artifacts here.

## Principles

- Prefer official plain-text snapshots.
- Preserve source URL and effective date with each stored rules artifact.
- Raw rules text is reference material until mapped into explicit simulator contracts.
- Store rules text snapshots at `raw/comprehensive_rules_<effective-date>.txt`.
- Store a provenance sidecar next to each text snapshot.
